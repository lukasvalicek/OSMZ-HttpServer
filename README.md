# OSMZ-HttpServer

/camera - obrázek, co 5s refresh (aktuálně je zakomentováno přepisování toho souboru)
/camera/snapshot - vždy při obnovení se natáhne nový obrázek z kamery
/camera/stream - stream z kamery (nový obrázek co 1s)

Příklady cgi-bin:
/cgi-bin/uptime
/cgi-bin/date
/cgi-bin/ls
/cgi-bin/ls OSMZ
